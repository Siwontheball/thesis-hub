# ThesisHub

社会学の研究室での使用を目的とした、論文アーカイブおよび進路管理プラットフォーム。

## プロジェクトの概要
論文のPDF管理だけでなく、卒業生の進路データも蓄積できるサイト。

## 主な機能
- **論文管理**: PDFのアップロード、一覧表示
- **全文検索**: タイトル、要約、キーワードからの部分一致検索
- **進路管理**: 卒業生の業界、職種、卒業年などの登録・閲覧

## 使用技術（Tech Stack）
- **Backend**: PHP 8.x / Laravel 11.x
- **Frontend**: Tailwind CSS / Vite
- **Database**: MySQL (AWS RDS)
- **Infrastructure**: AWS (S3, RDS)
